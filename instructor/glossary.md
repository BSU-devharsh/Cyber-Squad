# Glossary (acronyms expanded and defined)

- **AES** Advanced Encryption Standard. Strong modern encryption.
- **API** Application Programming Interface. How programs talk to each other.
- **CIA triad** Confidentiality, Integrity, Availability. The three goals of security.
- **CLI** Command Line Interface. Controlling a computer by typing commands.
- **CSRF** Cross-Site Request Forgery. Tricking a logged-in user into an unwanted request.
- **CTF** Capture The Flag. A security game of solving puzzles to find flags.
- **DNS** Domain Name System. Turns names into IP addresses.
- **EXIF** Exchangeable Image File Format. Hidden data inside photos.
- **HTTP / HTTPS** HyperText Transfer Protocol / HTTP Secure. The web's language; the S adds encryption.
- **IDS** Intrusion Detection System. Software that watches for attacks.
- **IP** Internet Protocol. The addressing system of the internet.
- **LLM** Large Language Model. An AI trained on text that follows instructions.
- **MFA / 2FA** Multi-Factor / Two-Factor Authentication. Two or more proofs of identity.
- **OS** Operating System. Windows, macOS, or Linux.
- **OSINT** Open-Source Intelligence. Investigation using only public information.
- **OWASP** Open Worldwide Application Security Project. Publishes the Top 10 web risks.
- **RSA** Rivest-Shamir-Adleman. A famous public-key encryption method.
- **SOC** Security Operations Center. The team that monitors and defends an organization.
- **SQL / SQLi** Structured Query Language / SQL Injection. Database language; an attack that abuses it.
- **SSH** Secure Shell. A secure way to log in to another computer.
- **TLS** Transport Layer Security. The encryption behind HTTPS.
- **VM** Virtual Machine. A computer that runs as software.
- **WAF** Web Application Firewall. Filters malicious web traffic.
- **XSS** Cross-Site Scripting. Running attacker code in a victim's browser.
