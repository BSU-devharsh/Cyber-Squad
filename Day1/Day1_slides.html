Cyber Squad - U.S. Cyber Range edition

Most of the internet runs on Linux, and security pros automate with Python. Today you build both foundations on a real Linux machine in the Cyber Range.

Morning kickoff: today's news

Linux runs the world: most web servers, Android phones, and supercomputers run Linux, so terminal skills are a real job requirement.
Automation is the job: a few lines of Python can scan a network or parse a log, which is why security teams rely on it.

What is a cyber range?

A cyber range is an isolated, cloud-based practice lab. You run real security tools without touching the public internet or any real victim. Everything stays inside the range, so you learn offense and defense legally and safely.

Cyber Range lab

Launch a Linux VM and use the terminal. Open the U.S. Cyber Range (uscyberrange.org) and your class environment, then follow the steps.

Lab steps

  1. Log in to the U.S. Cyber Range and open today's Linux environment from your class.
  2. In the terminal: pwd (where am I), ls -la (list files), cd and cat (move and read).
  3. Networking: ip addr shows your IP; ping -c 4 <in-range-host> checks a device (in-range only).
  4. Create a folder cybersquad and a file listing three commands you learned.

Stay safe and legal

Everything stays inside the isolated range. Scanning or attacking systems you do not own or have permission to test can violate the Computer Fraud and Abuse Act (CFAA).

Comments and print

Talking to the screen

print shows text; a # starts a comment Python ignores.

# a comment
print("Hello, Cyber Squad!")

Getting input

Reading from the user

input pauses for the user; it always returns text (str).

name = input("Name: ")
print("Welcome", name)

Making decisions

The if statement

if runs code only when a condition is True; else covers the rest.

if len(pw) < 12:
    print("Too short")
else:
    print("Good")

How an if statement decides

Illustration

The condition is checked once; exactly one branch runs.

Lists and indexing

Many values in order

Lists use square brackets; counting starts at 0.

ports=[22,80,443]
print(ports[0], len(ports))

While loops

Repeat until a condition changes

A while loop repeats while its condition is True. Make sure something changes.

n=0
while n<3:
    print(n); n=n+1

File permissions

Who can read, write, run

Linux files have read, write, and execute permissions for owner, group, and others. ls -l shows them; chmod changes them.
