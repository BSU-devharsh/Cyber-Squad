# BSU Cyber-Squad Camp

Five-day cybersecurity camp for Bowie State University students. No prior experience needed.

## What you will learn

| Day | Topic |
|-----|-------|
| Day 1 | Python basics and how the internet works |
| Day 2 | CIA triad, passwords, and account security |
| Day 3 | Cryptography: ciphers, hashing, and TLS |
| Day 4 | Web security: SQL injection, XSS, OWASP |
| Day 5 | CTF competitions, blue team, and career paths |

## How to use the materials

- Open your day slide deck at [bsu-devharsh.github.io/Cyber-Squad](https://bsu-devharsh.github.io/Cyber-Squad/)
- When you reach the Python exercise slide, click the Colab link to open your notebook (no install needed)
- For the Cyber Range lab, your instructor will give you login credentials to US Cyber Range
- Each day ends with a pair work session - work with your partner on the challenge in the student notebook

## What you need

A laptop and a browser. Everything else runs in the cloud.

## Cyber Range environments

- Days 1-2: Cyber Basics (2025.7)
- Days 3-5: K12 Cyber Security Concepts Using Kali Linux (2025.7)

Your instructor will share the direct links at the start of each lab session.
